# Room Planner AI

An intuitive interior design tool for creating 2D floor plans and visualizing them in real-time 3D with cloud synchronization, smart furniture snapping, and collision detection.

## Key Features
- **Cloud Sync**: Log in with Google to save and load projects anywhere.
- **2D/3D Parity**: Sketch in 2D, view in a dollhouse 3D mode.
- **Smart Dragging**: Real-time distance markers and wall snapping.
- **Export**: High-resolution PNG and print support.
