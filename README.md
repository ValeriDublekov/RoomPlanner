# Room Planner AI

An intuitive interior design tool for creating 2D floor plans and visualizing them in real-time 3D with cloud synchronization, smart furniture snapping, and collision detection. Room Planner AI transforms the complex task of room planning into a streamlined, visual, and highly accurate experience.

## Key Features
- **Blueprint-to-Vector Workflow**: Upload floor plans and use advanced Edge-Aware Snapping to trace walls accurately.
- **Cloud Sync**: Google Login via Firebase to save and load projects anywhere.
- **2D/3D Parity**: Sketch in 2D with precision tools, view in architectural 3D dollhouse mode.
- **Smart Dragging**: Real-time distance markers, wall snapping, and collision detection.
- **Precision Drafting**: Ortho mode, grid snapping, and image-edge detection tracing.
- **Custom Design**: Individual wall coloring, flooring textures, and advanced CAD furniture symbols.
- **High-Fidelity 3D**: Architectural sectional cutaways, clay-style rendering, and 1:1 coordinate translation.
- **Export**: High-resolution PNG export and professional print support.
