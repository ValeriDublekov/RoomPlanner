# Room Planner AI

An intuitive interior design tool for creating 2D floor plans and visualizing them in real-time 3D with cloud synchronization, smart furniture snapping, and collision detection.

## Key Features
- **Cloud Sync**: Google Login via Firebase to save and load projects anywhere.
- **2D/3D Parity**: Sketch in 2D with precision tools, view in architectural 3D dollhouse mode.
- **Smart Dragging**: Real-time distance markers, wall snapping, and collision detection.
- **Precision Drafting**: Ortho mode, grid snapping, and image-edge detection tracing.
- **Custom Design**: Individual wall coloring, flooring textures, and advanced CAD furniture symbols.
- **Export**: High-resolution PNG export and professional print support.
